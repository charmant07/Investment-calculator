## ðŸ§® Formula Used
For monthly contributions:
\[
FV = P(1 + r/n)^{nt} + PMT \left(\frac{(1 + r/n)^{nt} - 1}{r/n}\right)
\]

Where:  
- **P** = Initial investment  
- **PMT** = Monthly contribution  
- **r** = Annual interest rate (8%)  
- **t** = Number of years  
- **n** = 12 (months per year)
